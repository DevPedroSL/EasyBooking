# EasyBooking

EasyBooking es una aplicacion web para gestionar reservas en barberias. Permite a los clientes buscar barberias, reservar citas y consultar el estado de sus reservas. Tambien incluye paneles para barberos y administradores.

## Funcionalidades principales

- Busqueda de barberias por nombre y direccion.
- Vista de servicios con precio, duracion e imagenes.
- Reserva de citas mediante calendario de disponibilidad.
- Gestion de citas del usuario: ver detalles, cancelar citas pendientes y descargar PDF cuando la cita es aceptada.
- Notificaciones por correo cuando una cita es aceptada o rechazada.
- Gestion de perfil de usuario.
- Panel de barberia para horarios, servicios, agenda y citas.
- Panel de administracion para usuarios, barberias, solicitudes y copias de seguridad.

## Tecnologias

- Laravel 12
- PHP 8.2
- Blade
- Tailwind CSS
- Alpine.js
- Vite
- MySQL/MariaDB
